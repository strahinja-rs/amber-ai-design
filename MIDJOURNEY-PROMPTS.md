# Midjourney Prompts for Amber

## Prompting Guidelines

Based on current Midjourney V6/V7 best practices:

**Structure:** `[SUBJECT] + [STYLE/MOOD] + [LIGHTING] + [TECHNICAL] + [PARAMETERS]`

**Key Parameters:**
- `--ar` Aspect ratio (16:9, 1:1, 9:16, 4:5)
- `--s` Stylize (0-1000, default 100). Lower = literal, higher = artistic
- `--style raw` More literal interpretation
- `--no` Exclude elements

**Tips:**
- Be explicit, avoid filler words (no "award winning, 4k, 8k")
- Word order matters — important elements first
- Specify lighting precisely for commercial work

## Parameter Quick Reference

| Use Case | Recommended Settings |
|----------|---------------------|
| Product shots | `--s 50-75` `--style raw` for accuracy |
| Lifestyle | `--s 75-100` for natural warmth |
| Emotional scenes | `--s 100` for artistic feel |
| Abstract/Brand | `--s 100-150` for creativity |

| Format | Aspect Ratio |
|--------|-------------|
| Instagram Square | `--ar 1:1` |
| Instagram Story | `--ar 9:16` |
| Instagram Post | `--ar 4:5` |
| Website Hero | `--ar 16:9` |
| Facebook/LinkedIn | `--ar 16:9` or `--ar 1:1` |

## Negative Prompts (Always Include)

Add to product shots:
```
--no amber gemstone crystal rock mineral jewelry stone resin fossilized
```

Add to scenes:
```
--no blue cold clinical sterile stock generic circuit neural network AI robot futuristic neon purple harsh flash
```

---

## Iteration Tips

1. Start with base prompt, add `--s 50` for literal interpretation
2. If too literal, increase to `--s 100-150`
3. Use `--style raw` if prompt isn't being followed
4. Regenerate with `--chaos 20-40` for variety
5. Use multi-prompts with `::` to weight elements if needed
